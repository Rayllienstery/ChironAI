# UICollectionViewCompositionalLayout and section providers

Platform: iOS
Framework: UIKit
MinOS: iOS 15
Difficulty: intermediate
Concept Mode: all
RAG Strict: false

## Question

You build a complex dashboard screen using `UICollectionViewCompositionalLayout`
with multiple sections (grids, carousels, lists).
Explain:

- how section providers work and how they relate to the data source;
- how to configure different layouts per section;
- how to handle self-sizing cells and supplementary views correctly;
- common pitfalls (invalidation, layout crashes due to incorrect constraints).

## Expected Concepts

- UICollectionViewCompositionalLayout
- NSCollectionLayoutSection
- section provider
- self-sizing cells
- supplementary views
- invalidation

## RAG Requirement

The answer should align with Apple documentation on compositional layouts.

## Notes

Focuses on non-trivial UIKit layout composition.
