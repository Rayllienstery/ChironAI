Framework: SwiftUI
MinOS: iOS 15
Difficulty: advanced
Concept Mode: all

## Question

You create a screen where tapping an item expands it into a full-screen detail view with a smooth
animation of position and size. Explain:

- how to use `matchedGeometryEffect` to animate between two views in different parts of the hierarchy;
- how to manage the state that controls which item is expanded;
- how to avoid glitches when multiple items are tapped quickly;
- common pitfalls (ID mismatches, view identity, z-index issues).
