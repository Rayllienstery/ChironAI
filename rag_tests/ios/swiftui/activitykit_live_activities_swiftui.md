# Live Activities with ActivityKit and SwiftUI

Platform: iOS
Framework: SwiftUI
MinOS: iOS 16.1
Difficulty: advanced
Concept Mode: all
RAG Strict: true

## Question

You build a ride-tracking feature that uses Live Activities and SwiftUI.
Explain:

- how to define `ActivityAttributes` and dynamic `ContentState` for the ride;
- how to request, update and end a Live Activity using ActivityKit;
- how to design the SwiftUI view used for the Lock Screen and Dynamic Island;
- how background updates and app lifecycle interact with Live Activities.

## Expected Concepts

- ActivityKit
- Live Activity
- ActivityAttributes
- ContentState
- update(using:)
- end(dismissalPolicy:)

## RAG Requirement

The answer must reference ActivityKit and SwiftUI documentation for Live Activities.

## Notes

Checks non-trivial SwiftUI + system-feature integration knowledge.
