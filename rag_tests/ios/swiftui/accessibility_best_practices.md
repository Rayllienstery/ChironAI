Platform: iOS
Framework: SwiftUI
MinOS: iOS 15
Difficulty: intermediate
Concept Mode: all
RAG Strict: false

## Question

You want to ensure that a complex SwiftUI screen is fully accessible.
Explain:

- how to use `accessibilityLabel`, `accessibilityHint` and `accessibilityValue`;
- how to group or reorder elements for VoiceOver using `accessibilityElement(children:)`;
- how to support Dynamic Type and high contrast;
- common mistakes (missing labels on tappable elements, relying only on color).

## Expected Concepts

- accessibilityLabel
- accessibilityHint
- VoiceOver
- Dynamic Type
- accessibilityElement(children:)

## RAG Requirement

The answer should align with Apple accessibility documentation for SwiftUI.
